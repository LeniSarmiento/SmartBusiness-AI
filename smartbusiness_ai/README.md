# SmartBusiness AI

SmartBusiness AI es una plataforma web tipo SaaS para pequeños negocios. 
Permite controlar clientes, productos, ventas, inventario y visualizar reportes básicos para tomar mejores decisiones.

## Tecnologías usadas

### Frontend
- React
- Vite
- CSS moderno
- Axios
- Recharts

### Backend
- Python
- Django
- Django REST Framework
- SQLite para desarrollo

## Módulos principales

- Dashboard general
- Gestión de clientes
- Gestión de productos
- Registro de ventas
- Control de inventario
- Reportes inteligentes básicos

## Objetivo del proyecto

Este proyecto fue creado como una propuesta full stack para negocios pequeños que necesitan ordenar su información y visualizar sus ventas de manera sencilla.

## Instalación rápida

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
